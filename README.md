# Don Builds

> Retired July 2026. The current project collection now lives on the [Don Noel GitHub profile](https://github.com/donnoel).

Don Builds was a Next.js portfolio created to give a growing collection of apps, tools, and experiments one public home. It was deployed with AWS Amplify and served its purpose: learning the platform, shipping a complete web project, and establishing a clearer way to present the work.

The portfolio later moved closer to its source. Each project now owns a focused GitHub Pages site, while the GitHub profile provides the current collection and links. That structure is simpler to maintain and less likely to drift.

The original Next.js source remains here as a historical reference. This repository is no longer actively maintained or deployed to AWS.

## Historical Stack

- Next.js 16 and React 19
- TypeScript and CSS Modules
- AWS Amplify Hosting
- Data-driven project detail pages
- Responsive, accessible portfolio layouts
